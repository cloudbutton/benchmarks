CloudButton Serverless Benchmark


CloudButton Serverless Benchmark aims to become a general tool to compare Serverless offerings (Compute, Storage) in different public Cloud providers.

FLOPS Benchmark

CloudButton FLOPS benchmark evaluates compute power and scalability of cloud providers by running multiple compute intensive tasks concurrently. Benchmark source available here.

Storage Benchmark

CloudButton Storage benchmark evaluates the throughput of read and write operations to the object storage services of different cloud providers by running such operations in many tasks and concurrently. Benchmark source available here.
